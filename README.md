
# USD/INR Forex Trend Prediction using Machine Learning

## Project Overview
This project predicts USD/INR exchange rate trends using machine learning and time-series forecasting models.

## Models Used
- ARIMA
- Facebook Prophet
- LSTM Neural Network

## Dataset
Historical daily USD/INR exchange rate data (2015â€“2023).

## Data Source
Forex data is fetched dynamically using the yfinance library.

## Workflow
- Data preprocessing & visualization
- Model training & evaluation
- Performance comparison using RMSE & MAE
- Trend prediction visualization

## Results
LSTM achieved the highest predictive accuracy for volatile forex trends.

## Tech Stack
Python, Pandas, NumPy, Scikit-learn, TensorFlow, Prophet, Matplotlib.
